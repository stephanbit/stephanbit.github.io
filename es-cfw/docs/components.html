# Development Components

# Storybook

Storybook is a development environment for UI components. It allows you to browse a component library, view the different states of each component, and interactively develop and test components.

Benefits of living styleguides:

  • Less dublication of code and style
  • Accurate documentation of code and design standards
  • Useful for screenshot an isolated component testing
  • Communication tool between design and development
# Starts the storybook in watch mode
yarn serve:storybook

# Build a static html version, ready to deploy to webserver
yarn build:storybook
