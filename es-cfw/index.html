# ES Client Framework 3.0 alpha

This is a high scalable architecture and dev environment for new EASY Clients using Vue CLI 3.

# Features

  • Thorough Documentation: To quickly train new team members and consolidate knowledge we are using Vuepress for documentation and Storybook for component development.
  • Guaranteed Consistency: Opinionated linting for Vue, JavaScript/JSON, SCSS, and Markdown, integrated and run against staged files on pre-commit.
  • First-class Tests: Practice test-driven development with both unit and end-to-end tests. Unit tests with Jest live as first-class citizens alongside your source files, while Cypress provides reliable end-to-end tests in an intuitive GUI for development (Midnight for Selenium Server in progress).
  • Speedy Development: Between configurable generators, handy aliases, and global base components, your productivity will boost.

# Getting started

# 1. Clone the repository.
git clone  https://bitbucket.easy.de/scm/esc/es-cfw.git

# 2. Enter your newly-cloned folder.
cd es-cfw

# 3. Install dependencies. Make sure yarn is installed: https://yarnpkg.com/lang/en/docs/install
yarn

# 4. Read the documentation linked below for "Setup and development".
